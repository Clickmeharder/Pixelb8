Installation Guide

Because this is a specialized indie tool, Windows may flag it as "Unrecognized." Follow these steps to install:

  1. Download the suspect_setup.exe above.
  2. Run the installer. If a blue "Windows protected your PC" box appears:
    — Click "More Info"
    — Click "Run Anyway"
  3. The S.U.S.P.E.C.T Splash Screen will appear while the app initializes.
  4. Set your Entropia Log folder path and you're ready to hunt!
